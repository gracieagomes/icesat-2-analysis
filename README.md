# icesat-2-analysis
This is the README document for my summer research under Professor Joanmarie Del Vecchio. This research program has been split into two tasks divided under two research interns, the tasks are as follows: (1) Integrate the DISP product into analysis of topographic change in tundra landscapes in Alaska, and (2) Use the ICESat-2 product to assess vegetation structure and fire recovery in Siberian larch forest. For this project, my role is to work on task (2). I will be using VS Code, the WM astral sciclone, and Jupyter notebook to track my findings during this research.
